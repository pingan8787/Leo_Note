关于JS模块化的使用，大家肯定不陌生，现有的主要两种：CMD和AMD，还有一种兼容CMD和AMD写法的UMD。一般前端框架都支持AMD，而Node.js采用的是CMD的模块语法。

ES6中规范化了模块的导入和导出语法，在规范的15.2章节中。现在浏览器还都不支持，想要尝试的可以使用Traceur或者Babel。貌似Angular 2.0也要采用ES6的模块化语法，相信以后浏览器肯定会逐步支持的。

本篇文章只是大概介绍一下基本的语法，详细使用留在以后浏览器支持了 再说。


export——

基本的两种用法：

[javascript] view plain copy
  1. export function foo() {  
  2. // ..  
  3. }  
  4. export var awesome = 42;  
  5. var bar = [1,2,3];  
  6. export { bar };  

[javascript] view plain copy
  1. function foo() {  
  2. // ..  
  3. }  
  4. var awesome = 42;  
  5. var bar = [1,2,3];  
  6. export { foo, awesome, bar };  

导出的时候重命名:
[javascript] view plain copy
  1. function foo() { .. }  
  2. export { foo as bar };  

默认导出，每个模块只能有一个默认导出：
[javascript] view plain copy
  1. function foo(..) {  
  2. // ..  
  3. }  
  4. export default foo;  
  5. export{ foo as default };  

混合默认导出和普通的导出：
[javascript] view plain copy
  1. function foo() { .. }  
  2. function bar() { .. }  
  3. function baz() { .. }  
  4. export { foo as default, bar, baz, .. };  

从其他模块导出：
[javascript] view plain copy
  1. export { foo, bar } from "baz";  
  2. export { foo as FOO, bar as BAR } from "baz";  
  3. export * from "baz";  


import——

[javascript] view plain copy
  1. import { foo } from "foo";  
  2. foo();  
[javascript] view plain copy
  1. import { foo as theFooFunc } from "foo";  
  2. theFooFunc();  
[javascript] view plain copy
  1. import foo from "foo";  
  2. // or:  
  3. import { default as foo } from "foo";  

[javascript] view plain copy
  1. export default function foo() { .. }  
  2. export function bar() { .. }  
  3. export function baz() { .. }  
  4.   
  5. import FOOFN, { bar, baz as BAZ } from "foo";  
  6. FOOFN();  
  7. bar();  
  8. BAZ();  

[javascript] view plain copy
  1. export function bar() { .. }  
  2. export var x = 42;  
  3. export function baz() { .. }  
  4.   
  5. import * as foo from "foo";  
  6. foo.bar();  
  7. foo.x; // 42  
  8. foo.baz();  

import有一个hoisted的过程，和var声明变量一样：
[javascript] view plain copy
  1. foo();  
  2. import { foo } from "foo";
