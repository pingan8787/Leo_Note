  1. 代码： for (var i = 0; i < 10; i++) { setTimeout(function() { console.log(i); }, 100 * i); }
求执行结果.
错：1,2,3,4......10       （改正：用let 代替 var来声明变量）
对：10个10
原因： setTimeout在若干毫秒后执行一个函数，并且是在for循环结束后。 for循环结束后，i的值为10。 所以当函数被调用的时候，它会打印出 10！
就是for循环执行飞快，setTimeout要若干毫秒后执行。
