有时候想同时（同级）展示多个视图，而不是嵌套展示，例如创建一个布局，有 sidebar（侧导航） 和 main（主内容） 两个视图，这个时候命名视图就派上用场了。你可以在界面中拥有多个单独命名的视图，而不是只有一个单独的出口。如果 router-view 没有设置名字，那么默认为 default。

// 一个视图使用一个组件渲染，因此对于同个路由，多个视图就需要多个组件。确保正确使用 components 配置（带上 s）：
const Leo1 = { template: `<div><h1>Leo来了...</h1></div>` }
const Leo2 = { template: `<div><h2>Leo飞起来了...</h2></div>` }
const Leo3 = { template: `<div><h3>Leo开坦克...</h3></div>` }
const router5 = new VueRouter({
    mode: 'history',
    routes: [

        {
            path: '/',
            components: {
                default: Leo1,
                a: Leo2,
                b: Leo3
            }
        },
        {
            path: '/other',
            components: {
                default: Leo2,
                a: Leo1,
                b: Leo3
            }
        }

    ]
})
const app5 = new Vue({
    el: '#mytest5',
    router: router5
})
