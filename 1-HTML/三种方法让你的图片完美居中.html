三种方法让你的图片完美居中

创建时间：2016/11/15 星期二 15:53
更新时间：2017/2/10 星期五 10:45
标签：图片居中, 知乎收藏

一，table-cell + vertical-align: middle
<!DOCTYPE html>
<html lang="zh-cn">
<head>
 <meta charset="UTF-8">
 <title>Document</title>
 <style>
  #parent{
   display: table-cell;
   width: 200px;
   height: 200px;
   border: 1px solid black;
   vertical-align: middle;
   text-align: center;
  }
 </style>
</head>
<body>
 <div id="parent">
  <img src="echo.jpg" alt="小付"/>
 </div>
</body>
</html>

二，position定位
<!DOCTYPE html>
<html lang="zh-cn">
<head>
 <meta charset="UTF-8">
 <title>Document</title>
 <style>
  #parent{
   position: relative;
   width: 200px;
   height: 200px;
   border: 1px solid black;
  }
  img{
   position: absolute;
   left: 50%;
   top: 50%;
   transform: translate(-50%,-50%);
  }
 </style>
</head>
<body>
 <div id="parent">
  <img src="echo.jpg" alt="小付"/>
 </div>
</body>
</html>

三，flex弹性盒模型
<!DOCTYPE html>
<html lang="zh-cn">
<head>
 <meta charset="UTF-8">
 <title>Document</title>
 <style>
  #parent{
   width: 200px;
   height: 200px;
   border: 1px solid black;
   justify-content: center;
   align-items: center;
   display: flex;
  }
 </style>
</head>
<body>
 <div id="parent">
  <img src="echo.jpg" alt="小付"/>
 </div>
</body>
</html>

注：1.这三种方法可以用于子元素为行内元素和块状元素；
       2.建议使用第二种，简单，好用，重要的还是看你的具体情况；
       3.最重要的一点：图片无法显示，我也没办法， ；
